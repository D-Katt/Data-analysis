Анализ ретроспективных данных по основным параметрам работы нефтяной скважины:
- Формирование графиков с динамикой ключевых параметров;
- Формирование графиков зависимости между параметрами;
- Формирование матрицы корреляции (heatmap).

Построение прогнозов методом подбора оптимальным параметров уравнений с использованием инструмента curve_fit() библиотеки scipy.optimize:
- Прогноз дебита нефти по гиперболической кривой падения;
- Прогноз обводненности по экспоненциальной кривой роста.
