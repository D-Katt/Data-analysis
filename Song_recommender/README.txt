Код содержит алгоритмы для рекомендации пользователям музыкальных композиций на основе общего рейтинга популярности песен
и индивидуальных предпочтений позователей, определеяемых на основе предшествующей истории прослушиваний.
Алгоритмы реализованы в формате класса Recommender() с несколькими методами, позволяющими формировать списки
рекомендуемых песен с учетом различных параметров, характеризующих пользователя: список прослушанных песен и исполнителей, период выхода прослушанных песен, альбомы, в которые они входят, список пользователей с похожими предпочтениями и прослушанные ими песни.

Источники данных для анализа:
 1. База данных, включающая 3 параметра:
    - ID пользователя;
    - ID песни;
    - количество прослушиваний).
        Ссылка: https://static.turi.com/datasets/millionsong/10000.txt
2. База данных, включающая параметров:
   - song_id;
   - title;
   - release;
   - artist_name;
   - year.
       Ссылка: https://static.turi.com/datasets/millionsong/song_data.csv
