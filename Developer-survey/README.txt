Анализ результатов ежегодного опроса Stack Overflow Developer Survey за 2019 год.

Источник базы данных: https://insights.stackoverflow.com/survey

Компоненты базы данных:
- survey_results_public.csv - основные результаты опроса (по одному респонденту на строку, по одной колонке на вопрос)
- survey_results_schema.csv - перечень вопросов, которые соответствуют каждому столбцу из предыдущей таблицы
